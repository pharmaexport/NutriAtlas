# Methodology

## Core principles

NutriAtlas is built around source traceability.

Every nutrition value must be linked to an institutional source.

## Data flow

1. Store raw institutional data.
2. Validate source metadata.
3. Normalize food and nutrient fields.
4. Preserve original source references.
5. Expose values through the API and user interface.

## Rules

- No value without a source.
- No recommendation without a citation.
- No commercial content inside the scientific data layer.
- All imported datasets must be versioned.
