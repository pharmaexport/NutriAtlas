# Roadmap

## Phase 1 - Data foundation

- Document CIQUAL as master source.
- Prepare raw data storage.
- Create metadata model.
- Build CIQUAL importer.

## Phase 2 - API

- Food search endpoint.
- Food details endpoint.
- Nutrient listing endpoint.
- Comparison endpoint.

## Phase 3 - Product

- Public web interface.
- Food comparison pages.
- Nutrient ranking pages.
- Citation display on every value.

## Phase 4 - Expansion

- Add USDA FoodData Central as international benchmark.
- Add EFSA references for risk and safety context.
- Add scientific documentation pages.
